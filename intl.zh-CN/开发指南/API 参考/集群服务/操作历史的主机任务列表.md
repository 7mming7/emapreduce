# 操作历史的主机任务列表 {#concept_pdt_thm_2gb .concept}

查询指定的操作历史中，主机对应的任务列表信息

## 请求参数 {#section_obj_yxl_2gb .section}

|参数|类型|是否必选|示例值|描述|
|--|--|----|---|--|
|ClusterId|String|是|C-F32FB31D82954C64|集群 ID|
|HostId|String|是|11|主机 ID|
|RegionId|String|是|cn-hangzhou|RegionID|
|PageNumber|Integer|否|1|分页查询的查询页码|
|PageSize|Integer|否|100|分页查询的每页记录数|
|Status|String|否|SUCCESS|执行状态|
|AccessKeyId|String|否|\*\*\*\*|阿里云 AccessKey ID 信息|

## 返回参数 {#section_vbj_yxl_2gb .section}

|字段|类型|示例值|描述|
|--|--|---|--|
|RequestId|String|DF202AC2-5D5D-4288-B608-B7B1595B5C7C|请求ID|
|TotalCount|Integer|1|记录总数|
|PageNumber|Integer|1|当前的页号|
|PageSize|Integer|100|每页的记录数|
|ClusterOperationHostTaskList| | |操作历史中，主机上的任务列表信息|
|TaskId|String|111|任务ID|
|TaskName|String|START\_SparkHistory\_ON\_emr-header-1|任务名称|
|Status|String|SUCCESS|执行状态|
|Percentage|String|100|执行进度（百分比）|

    ```
    &OperationId=11
    &RegionId=cn-hangzhou
    &AccessKeyId=****
    &PageNumber=1
    &PageSize=100
    &Status=SUCCESS
    &<公共请求参数>
    ```
