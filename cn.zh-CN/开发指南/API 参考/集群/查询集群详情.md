# 查询集群详情 {#concept_gcm_ydb_kfb .concept}

查询集群的基本信息，包括：付费、ECS 机器概况、E-MapReduce 服务服务列表等。

## 请求参数 {#section_u3c_m2b_kfb .section}

|参数|类型|是否必须|示例值|描述|
|--|--|----|---|--|
|Id|String|是|C-E331B8AC12BF5DB4|集群 ID|
|RegionId|String|是|cn-hangzhou|区域|
|AccessKeyId|String|否|\*\*\*\*|AccessKeyId|
