# DescribeEmrMainVersion {#concept_h1c_kkq_dgb .concept}

You can call this operation to view the details of an EMR version.

## Request parameters {#section_pdw_lv4_dgb .section}

|Parameter|Type|Required|Example|Description|
|---------|----|--------|-------|-----------|
|RegionId|String|Yes|cn-hangzhou|The region ID.|
|AccessKeyId|String|Yes|LTAI8ljWyu7y\*\*\*\*|The AccessKey ID.|
|EmrVersion|String|No|EMR-3.15.0|The version of EMR.|

## Response parameters {#section_vdw_lv4_dgb .section}

|Parameter|Type|Example|Description|
|---------|----|-------|-----------|
|RequestId|String|BF4FBAC6-B03E-4BFB-B6DB-EB53C34F2E22|The ID of the request.|
|EmrMainVersion| | |The details of the EMR version.|
|RegionId|String|cn-hangzhou|The region ID.|
|EmrVersion|String|EMR-3.15.0|The version.|
|EcmVersion|Boolean|true|
|ImageId|String|m-546h5ytodhmtrohm54ortb|The ID of the image.|
|Display|Boolean|True|Indicates whether the version is open to the public.|
|StackName|String|EMR|The name of the software stack.|
|StackVersion|String|0|A reserved parameter.|
|ClusterTypeInfoList| | |The list of information of cluster types.|
|ClusterType|String|HADOOP|The type of the cluster.|
|ServiceInfoList| | |The list of service information.|
|ServiceName|String|HIVE|The name of the service.|
|ServiceDisplayName|String|Hive|The display name of the service.|
|ServiceVersion|String|2.3.3-1.0.2|The internal version of the service.|
|ServiceDisplayVersion|String|2.3.3|The display version of the service.|
|Mandatory|Boolean|true|Indicates whether the service is required.|
|Display|Boolean|true|Indicates whether the service is displayed.|
|WhiteUserList|String|\[""\]|A reserved parameter.|
